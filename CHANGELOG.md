# Changelog

All notable changes to this plugin will be documented in this file.

## [0.3.0] - 2026-03-19

### Added
- MCP server registration: writes `.mcp.json` in project root on startup and file open
- Adds `.mcp.json` to `.gitignore` if not already listed
- Cleans up `.mcp.json` on shutdown
- Tracks multiple project roots across tab switches

## [0.2.0] - 2026-03-18

### Added
- Initial release
- Opens Claude Code CLI in embedded terminal via Plugins menu (Ctrl+Shift+I)
